# Anime Hair Supporter for Maya

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

アニメ調キャラクターの髪の毛（ポリゴン・リボン・ボーン）のモデリングからリギングまでを強力にサポートする、Autodesk Maya向けのスクリプトツール一式です。

選択したエッジからガイドカーブをダイレクトに生成し、滑らかな太さ変化を持つリボン状のサーフェス（肉付け）の作成、そして毛先に向かって正しい親子関係を持つジョイント（骨）の自動配置までを直感的なUIで行うことができます。

---

## 特徴 (Features)

1. **選択エッジ完全追従型のカーブ生成**
   - ポリゴンメッシュのトランスフォーム値を完全に無視し、選択されたエッジ（コンポーネント）の絶対ワールド座標をダイレクトに取得して、真上にぴったり沿う3次カーブを生成します。
2. **肉付け・シルエットコントロール**
   - 2次ベジェ曲線による滑らかなカーブ補間を導入。「根元」「中間」「毛先」の3点スライダーにより、「根元を細く、中間を太く、毛先を尖らせる」といったアニメ髪特有の自由なシルエット調整が可能です。
   - 断面の厚みを完全にゼロにした「1枚の帯（リボン状のサーフェス）」を生成します。
3. **ジョイント自動生成**
   - カーブが描かれた方向に関わらず、頭頂部（+Y）から毛先（-Y）に向かって自動的にソートし、正しい親子関係（上が親、下が子）で綺麗な骨のチェインを配置します。
4. **全自動メニュー追加システム**
   - 難しいコードのコピペやスクリプトエディタでの実行は一切不要です。Mayaの起動時にトップメニューへ自動的にツールの実行メニューが追加されます。

---

## インストール方法 (Installation)

本ツールは「全自動メニュー追加システム」を採用しているため、簡単なファイルの配置だけで導入が完了します。

1. ダウンロードしたリポジトリ（ZIPファイルなど）を解凍します。
2. 中にある以下の **1つのファイル** と **1つのフォルダ** を選択します。
   - `AnimeHairSupporter_Maya.mod` （ファイル）
   - `AnimeHairSupporter_Maya` （フォルダ）
3. 上記の2つを、お使いの環境の以下のフォルダ内へコピー（配置）してください。
   - **Windowsの場合:** `C:\Users\<ユーザー名>\Documents\maya\modules\`
   - ※ `modules` フォルダがない場合は、新規作成してください。
4. 配置後、Autodesk Mayaを起動（または再起動）します。
5. 画面上のトップメニュー（「ファイル」「編集」などの並び）に、自動的に **[AnimeHairSupporter]** というメニューが追加されます。メニュー内の項目をクリックすることでツールが起動します。
